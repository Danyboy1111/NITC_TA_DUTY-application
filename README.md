# NITC_TA_DUTY-application

A simple application to manage Teaching Assistant (TA) duty assignments for NITC. This repository contains the code and configuration needed to track, assign, and notify TAs about their duty schedules.

## Features

- Create and manage TA duty schedules
- Assign TAs to duty slots
- View duty roster and history
- Simple configuration for notifications
